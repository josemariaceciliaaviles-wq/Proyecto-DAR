## Instrucciones de ejecución

Para ejecutar la aplicación, primero hay que iniciar el servidor RMI.

Por defecto, la aplicación utiliza la siguiente configuración:

IP del servidor: 192.168.0.10  
Puerto: 1099

Después, se debe ejecutar el cliente. Al iniciar el cliente, se pedirá introducir la IP y el puerto del servidor.

En el caso del puerto, se puede escribir manualmente 1099 o simplemente pulsar Enter. Si se pulsa Enter sin introducir ningún valor, la aplicación utilizará automáticamente el puerto por defecto: 1099.

Una vez establecida la conexión con el servidor, aparecerá un menú por consola con todas las opciones disponibles de la aplicación.

Para acceder a la aplicación se pueden usar las siguientes credenciales:

Usuario: admin  
Contraseña: finisterre

Desde el menú se podrán realizar las operaciones implementadas, como iniciar sesión, cerrar sesión, crear un envío, consultar un envío, actualizar un envío o listar todos los envíos.

### Pasos de ejecución

1. Ejecutar el servidor.
2. Ejecutar el cliente.
3. Introducir la IP del servidor: 192.168.0.10
4. Introducir el puerto: 1099
   - También se puede pulsar Enter para usar el puerto por defecto.
5. Iniciar sesión con:
   - Usuario: admin
   - Contraseña: finisterre
6. Utilizar el menú de opciones que aparece por consola.
